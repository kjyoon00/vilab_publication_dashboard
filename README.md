# KAIST AI Research Dashboard

KAIST의 현재 AI 연구 과제와 추진 현황을 한눈에 보기 위한 정적 대시보드 프로토타입입니다.

## 실행 방법

`/Users/kjyoon/Documents/함해보자/index.html` 파일을 브라우저에서 열면 KAIST AI 과제 대시보드를 바로 볼 수 있습니다.

`/Users/kjyoon/Documents/함해보자/index1.html` 파일을 열면 VILab 논문 대시보드를 볼 수 있습니다.

## 포함된 내용

- KAIST 공식 뉴스와 연구 소개 페이지를 바탕으로 정리한 공개형 과제 데이터
- 상태 필터, 분야 필터, 검색
- 카드형 뷰, 최근 업데이트 타임라인, 테이블 뷰
- VILab 공개 논문 목록 기반 논문 대시보드
- 논문 대시보드의 빠른 venue 필터, 저자 집계, 저자 클릭 필터, 특허 제외 보기, 논문/특허 분리 탭

## 주의

- 현재 데이터는 공개 자료를 바탕으로 수동 정리한 샘플입니다.
- 실제 운영용 대시보드로 확장하려면 KAIST 내부 과제관리 데이터 또는 공공 R&D API 연동이 필요합니다.
